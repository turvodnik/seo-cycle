# ORM — управление репутацией и отзывами

Отзывы влияют на локальное ранжирование (карты), конверсию и доверие. `review-velocity.py`
даёт план набора отзывов; этот модуль — про **мониторинг и реакцию**.

## Что мониторить (раз в неделю/месяц)
- Новые отзывы и оценка на **Яндекс.Картах**, **2ГИС**, **Google Business**, **Яндекс.Маркете**, отзовиках (Отзовик).
- Динамика средней оценки; всплески негатива.
- Упоминания бренда вне карточек (соцсети, форумы) — brand mentions.

## Алерт на негатив
Свежий негатив (оценка ≤ 3) — реагировать быстро (24-48ч). Отправить уведомление:
```bash
python3 ~/.claude/skills/seo-cycle/scripts/notify.py \
  "Негативный отзыв на <платформа>: <суть>. Ответить за 24-48ч." \
  --title "ORM: негатив" --level alert
```
(Сбор отзывов — браузером/вручную или API площадки; notify шлёт в Telegram.)

## Работа с отзывами
- На негатив — спокойный публичный ответ + решение проблемы (это видят будущие клиенты и поисковик).
- На позитив — благодарность, мягко стимулировать ещё (QR на складе, follow-up после доставки).
- НЕ накручивать — Яндекс/2ГИС жёстко фильтруют, бан-риск.

## Выход
`<cycle>/orm/<date>.md` — новые отзывы по площадкам, оценка, негатив + статус ответа. Отставание по объёму → `review-velocity.py`; задачи → Phase 10.
