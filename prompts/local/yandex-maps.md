# Яндекс.Карты / Яндекс.Бизнес / 2ГИС — тактики локального SEO (РФ-приоритет)

Те же 5 тактик, что и для Google, но под РФ-экосистему. Для России Яндекс.Карты + 2ГИС обычно дают больше локального трафика, чем Google Maps. Источник — `business_profile` (`yandex_business_url`, `2gis_url`, `competitors[].yandex`). Выполнять через Claude in Chrome MCP / browser-skill. Делегат: `yandex-seo-specialist`.

---

## 1. Аудит рубрик конкурентов (быстрая победа)
```
Открой Яндекс.Карты, поиск по 3 целевым ключам (регион из locale). Для конкурентов в выдаче
зайди в карточки, извлеки ВСЕ рубрики (основная + дополнительные). То же в 2ГИС.
Таблица: рубрика | конкурент A/B/C | есть ли у меня (Яндекс / 2ГИС). Выдели недостающие рубрики.
```
Результат → `<cycle>/local/yandex-categories-gap.md`. Действие: добавить рубрики в Яндекс.Бизнес и 2ГИС. Учесть: рубрики Яндекса и 2ГИС различаются — проверять обе.

## 2. Скорость отзывов (Яндекс.Карты + 2ГИС) и план догона
```
Открой карточки топ-3 конкурентов в Яндекс.Картах и 2ГИС. Посчитай отзывы за 30/60/90 дней
по каждой платформе отдельно, текущее число и оценку. Яндекс учитывает свежесть и развёрнутость
отзывов — фиксируй и это.
```
Расчёт догона:
```bash
python3 ~/.codex/skills/seo-cycle/scripts/review-velocity.py \
  --platform yandex --my-total <N> --leader-total <M> --leader-30d <X> --my-target-30d <Y>
```
Результат → `<cycle>/local/yandex-reviews-plan.md`. Важно: не накручивать — Яндекс агрессивно фильтрует фейк; стимулировать реальные отзывы (QR на складе, после доставки).

## 3. Календарь «Новостей»/постов Яндекс.Бизнеса
```
Проверь раздел Новости/Акции в Яндекс.Бизнесе топ-3 конкурентов (и витрину товаров): частота,
типы, темы. Составь 8-недельный календарь, 2-3 публикации/нед: готовый текст, идея фото,
локальный ключ. Учесть формат витрины/акций Яндекс.Бизнеса.
```
Результат → `<cycle>/local/yandex-posts-calendar.md`.

## 4. Визуальное доминирование (фото Яндекс.Карты + 2ГИС)
```
Проанализируй фото конкурентов в Яндекс.Картах и 2ГИС: количество, типы (интерьер/экстерьер/
товар/процесс/команда), панорамы, качество. 30-дневный план загрузки: типы фото, подписи,
ключи. Для emwoody — реальные фото склада/товара (без стоков).
```
Результат → `<cycle>/local/yandex-visual-plan.md`.

## 5. Локальная видимость («Как клиенты вас находят»)
```
В Яндекс.Бизнесе открой статистику → «Запросы для перехода» / «Как вас находят» — реальные
запросы, по которым видят карточку. Сопоставь с целевыми ключами: какие закрыты, какие нет.
Проверь позиции карточки по районам региона (для МО — Мытищи/Одинцово/Подольск/Химки).
```
Результат → `<cycle>/local/yandex-visibility.md`. У Яндекса нет grid-API как у Google — оцениваем через статистику Бизнеса + ручную проверку по районам.
