# Local-pack — локальное SEO для карт (Google + Яндекс/2ГИС)

Набор воспроизводимых тактик локального продвижения для бизнеса с офлайн-точкой
или зоной обслуживания. **Парный подход:** каждая тактика — для Google Maps/GBP
(`google-maps.md`) И для Яндекс.Карт/Бизнеса/2ГИС (`yandex-maps.md`). Для РФ
приоритет — Яндекс + 2ГИС.

## Когда использовать
Локальный бизнес, e-commerce со складом/самовывозом, услуги в регионе. Phase 1
(аудит локального доминирования) и Phase 9-10 (мониторинг/догон конкурентов).

## Как запускать
- **Claude Code:** через Claude in Chrome MCP (`mcp__Claude_in_Chrome__*`) — открыть карты, снять данные.
- **Codex:** через нативный `browser`/`playwright` skill (см. `docs/codex-runtime.md`).
- Часть тактик (скорость отзывов) считает детерминированно `scripts/review-velocity.py`.

## Источник данных о бизнесе и конкурентах
Берётся из `business_profile` в `seo-cycle.yaml`:
- `gbp_url`, `yandex_business_url`, `2gis_url` — свои профили;
- `competitors: [{name, gbp, yandex}]` — топ-3 конкурента (идея «постоянного профиля» — не спрашивать каждый раз).

## 5 тактик (в обоих файлах)
1. **Категории/рубрики** — gap-анализ: какие категории есть у конкурентов, но не у вас (быстрая победа).
2. **Скорость отзывов** — темп конкурентов за 30/60/90 дней → план догона (`review-velocity.py`).
3. **Календарь публикаций** — что и как часто постят конкуренты → 8-недельный план постов.
4. **Визуальное доминирование** — аудит фото конкурентов по типам → 30-дневный план загрузки.
5. **Локальная видимость** — где вы ранжируетесь на карте по целевым ключам (Google geo-grid / Яндекс «как находят»).
