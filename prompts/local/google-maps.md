# Google Maps / Business Profile — тактики локального SEO

Источник бизнеса и конкурентов — `business_profile` в `seo-cycle.yaml` (`gbp_url`, `competitors[].gbp`, `area_served`, целевые ключи). Выполнять через Claude in Chrome MCP (Claude) или browser-skill (Codex). Не выдумывать данные — снимать со страниц.

---

## 1. Аудит категорий конкурентов (быстрая победа)
```
Открой Google Карты и выполни поиск по моим 3 целевым ключам (из business_profile / целевых ключей).
Для каждого поиска: какие конкуренты в выдаче Карт. Зайди на их GBP-страницы, извлеки ВСЕ категории
(основную + дополнительные). Сведи в таблицу: категория | у конкурента A/B/C | есть ли у меня.
Чётко выдели категории, которые есть у конкурентов, но НЕ у меня — это кандидаты на добавление.
Приоритизируй по релевантности моему бизнесу.
```
Результат → `<cycle>/local/google-categories-gap.md`. Действие: добавить 1-2 релевантные вторичные категории в GBP.

## 2. Скорость отзывов и план догона
```
Открой GBP топ-3 конкурентов. Для каждого посчитай отзывы за последние 30 / 60 / 90 дней
(скорость получения). Зафиксируй текущее общее число отзывов и среднюю оценку.
```
Затем посчитай план догона детерминированно:
```bash
python3 ~/.claude/skills/seo-cycle/scripts/review-velocity.py \
  --my-total <N> --leader-total <M> --leader-30d <X> --my-target-30d <Y>
```
Результат → `<cycle>/local/google-reviews-plan.md`: сколько отзывов/мес нужно и срок догона.

## 3. Календарь публикаций GBP (8 недель)
```
Проверь раздел «Публикации» (Posts) GBP топ-3 конкурентов: как часто постят, типы постов
(новость / акция / товар / событие), темы. Затем составь мне 8-недельный календарь, 2-3 поста/нед.
Для каждого поста: готовый текст, идея изображения, целевой локальный ключ, тип поста.
```
Результат → `<cycle>/local/google-posts-calendar.md`. (Большинство конкурентов почти не постят — окно.)

## 4. Визуальное доминирование (фото)
```
Проанализируй фото/видео GBP топ-3 конкурентов: сколько фото, типы (интерьер, экстерьер, команда,
процесс работы, до/после, товар), уровень качества. Составь мне 30-дневный план загрузки:
точные типы фото, описания, ключи для имён файлов и подписей. Привяжи к area_served и целевым ключам.
```
Результат → `<cycle>/local/google-visual-plan.md`.

## 5. Локальная видимость (geo-grid)
```
Проверь, как я ранжируюсь в Google Картах по целевым ключам из разных точек area_served
(центр + края зоны обслуживания). Где я в топ-3, где проваливаюсь. Сравни с топ-3 конкурентами.
```
Альтернатива/усиление — плагин `seo-maps` (geo-grid rank tracking, Share of Local Voice). Результат → `<cycle>/local/google-visibility.md`.
