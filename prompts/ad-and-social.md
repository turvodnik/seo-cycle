# Реклама конкурентов + генерация объявлений и соцпостов (РФ)

Два связанных процесса: (1) разведка платной выдачи и соцактивности конкурентов,
(2) генерация наших объявлений и постов под РФ-каналы. Контент-логика — через плагин
`marketing-skills` с РФ-адаптацией (`docs/marketing-bridges.md`). Источник конкурентов —
`business_profile.competitors` + `target_local_keywords`.

## Часть A — разведка рекламы и соцсетей конкурентов

### Реклама (кто и что крутит по нашим ключам)
- **SpyFu** (запад, PPC — его сила): `spyfu-fetch.py domain-stats <competitor> --cc US` → их объявления, бюджет, paid-клики.
- **Serpstat**: рекламные объявления/история по ключам (РФ через `g_ru`).
- **Яндекс.Директ** (главное для РФ): вручную/браузером — поиск по `target_local_keywords`, кто в рекламных блоках, тексты объявлений, быстрые ссылки, цены в объявлениях.
- Свести: ключ | конкуренты в рекламе | их УТП/оффер | посадочная.

### Соцсети
- VK / Telegram / Дзен конкурентов: частота, форматы, темы, вовлечённость, что заходит.
- Свести → идеи и бенчмарк частоты (для `competitor-benchmark.py`: posts_per_month).

## Часть B — генерация (через marketing-skills + РФ-адаптация)

### Объявления (Яндекс.Директ / VK Реклама)
Делегат: `marketing-skills:paid-ads` + `marketing-skills:ad-creative` — **с заменой каналов** (Google Ads → Яндекс.Директ, Meta → VK Реклама; см. rf_channel_map). Для каждой группы ключей: заголовки (под лимиты Директа), тексты, быстрые ссылки, УТП на основе stock-first и реальных преимуществ (склад в Мытищах, доставка по МО, распил). Без эпитетов (tone из CLAUDE.md).

### Посты для платформ
- **Соцсети** (VK / Telegram / Дзен): `marketing-skills:social-content` РФ-адаптированный — контент-план 4-8 недель, тексты, идеи визуалов. Связать с блогом (анонсы статей) и акциями.
- **Карты-посты** (Google Business / Яндекс.Бизнес Новости): уже в `prompts/local/` (календарь постов) — переиспользовать.

## Выход
- `<cycle>/ad-social/competitor-ads.md` — разведка рекламы.
- `<cycle>/ad-social/social-calendar.md` — контент-план постов (VK/TG/Дзен + карты).
- `<cycle>/ad-social/ads-drafts.md` — черновики объявлений Директ/VK.
Бенчмарк частоты постов → `competitor-benchmark.py`; приоритет идей → `ice-score.py`.

## РФ-нюанс
Каналы — всегда РФ-стек (Директ/VK/TG/Дзен), не Google Ads/Meta. Учитывать закон о рекламе РФ и маркировку рекламы (ОРД/токены) для платных размещений.
