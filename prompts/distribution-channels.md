# Каналы дистрибуции (РФ) + товарные фиды / маркетплейсы

Контент и товар не должны жить только на своём сайте. Этот модуль — план выхода
в дополнительные РФ-каналы. Контент-логика — через `marketing-skills` с РФ-адаптацией.
Релевантность каналов зависит от проекта (e-commerce vs услуги vs контент).

## Онлайн-каналы дистрибуции (РФ)

| Канал | Когда | Через что |
|---|---|---|
| **Email-рассылки** | повторные продажи, опт, прогрев | `marketing-skills:email-sequence` + РФ ESP (Unisender/Sendsay/DashaMail). Сегменты: частники / бригады / закупщики |
| **Telegram-канал + бот** | B2B, лояльность, анонсы, прайс | контент-план постов; бот для заявок/прайса |
| **VK / Дзен** | охват, SEO-дубль контента, статьи | `marketing-skills:social-content`; кросс-постинг блога |
| **Видео (VK Видео / RuTube / YouTube / Дзен)** | «как выбрать/смонтировать», обзоры товара | `marketing-skills:video`; видео-отзывы = сильный E-E-A-T/trust |
| **Анонсы блога в каналах** | дистрибуция каждой статьи | связать с Phase 7 (после публикации → пост в VK/TG) |

## Товарные фиды / маркетплейсы (для e-commerce)

Если продаёшь товар — карточки должны быть и в товарных/прайс-площадках:

| Площадка | Что нужно | Примечание |
|---|---|---|
| **Яндекс.Маркет** | YML-фид (Яндекс.Вебмастер/Маркет), карточки, цены, наличие | главный РФ товарный канал; связь с Яндекс.Бизнес |
| **Озон / Wildberries** | фид/карточки по требованиям площадки | если модель позволяет (маржа, логистика) |
| **Google Merchant Center** | Google Shopping фид (GTIN, цена, наличие) | для Google Shopping; в РФ вторично после Маркета |
| **Профильные РФ-агрегаторы стройматериалов** | прайс/выгрузка | ниша emwoody — отраслевые каталоги |

Подготовка фида: корректный `Product` schema на карточках (у нас есть), GTIN/артикул,
актуальные цена/наличие из Woo, фото. Качество карточки = ранжирование в товарном поиске.

## РФ-нюансы
- Маркировка рекламы (ОРД/токены) для платных размещений.
- Email — согласие на рассылку (152-ФЗ), двойной opt-in.
- Видео-отзывы и реальные фото склада/товара (без стоков) — сильнее работают в РФ-нише.

## Выход
