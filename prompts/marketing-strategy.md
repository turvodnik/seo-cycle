# Маркетинг-стратегия + решение «нужна ли реклама»

Верхний слой над SEO-циклом: от целей бизнеса к плану каналов с бюджетом и KPI.
Главный вопрос — **нужна ли платная реклама или хватит органики + локалки**.
Решается на цифрах через `roi-calc.py`, а не на ощущениях.

## Шаги

1. **Цели и экономика.** Сформулируй цель в деньгах/заказах (не в «трафике»): сколько заказов/мес нужно, средний чек, маржа, допустимый CAC. Для emwoody — B2C + B2B (опт), у опта свой чек/маржа.

2. **Текущая картина по каналам.** Собери воронку (трафик→лиды→заказы→выручка) по каналам в CSV и прогони:
   ```bash
   python3 ~/.codex/skills/seo-cycle/scripts/roi-calc.py funnel.csv --margin <доля_маржи>
   ```
   Получишь ROI/ДРР/CAC по каналам и вердикт об окупаемости.

3. **Достаточно ли органики?** Сравни план по заказам с тем, что дают органика + локалка (карты). Если закрывают цель при целевом CAC — **реклама не нужна**, вкладывайся в SEO-цикл + локалку + контент. Если есть дефицит объёма — переходи к п.4.

4. **Если реклама нужна — медиаплан:**
   - Каналы РФ по приоритету: **Яндекс.Директ** (Поиск+РСЯ), **VK Реклама**, Telegram Ads, ретаргетинг. (НЕ Google Ads/Meta — см. `docs/marketing-bridges.md`.)
   - Бюджет от unit-экономики: при целевом CAC и плане заказов → бюджет = заказы × CAC. Проверь, что CAC < маржа с заказа.
   - Прогноз: ожидаемый ROI/ДРР по каждому каналу до запуска (по бенчмаркам ниши/конкурентов из SpyFu/Serpstat).
   - Тест-бюджет на 2-4 недели → замер `roi-calc.py` → масштабировать прибыльное, резать убыточное.

5. **Микс каналов (полноценный маркетинг):** органика (SEO-цикл) + локалка (карты) + контент/соцсети (дистрибуция) + реклама (если ROI>0) + email/ORM (удержание). Связать в `prompts/marketing-calendar.md`.

6. **KPI и пересмотр.** На канал — целевые CAC/ДРР/ROI; ежемесячный пересмотр (Phase 9-10) с `roi-calc.py`.

## Выход
`<cycle>/marketing-strategy.md` — цели → вердикт по рекламе → медиаплан/бюджет (если нужна) → микс каналов → KPI. Решения приоритизируй через `ice-score.py`.

## Принцип
Реклама — не самоцель, а инструмент при дефиците объёма с положительным ROI. Сначала выжать почти бесплатные органику + локалку (у них обычно лучший ROI), платку добавлять точечно под измеримый возврат.
